Projects
Forever Clothing Ecommerce
Technologies
This project is a full-stack eCommerce website built using the MERN stack — React.js, Node.js, Express.js, and MongoDB — with deployment on Vercel. The platform allows users to explore products, filter and sort results, select product variants (such as size), add items to the cart, and place orders by providing delivery details. Customers can choose between Cash on Delivery or secure online payments via integrated Stripe and Razorpay gateways. The project also features a robust admin dashboard, enabling administrators to manage the store — including adding, updating, and deleting products, as well as viewing all listed items and orders. The backend is powered by Node.js and Express.js APIs, while all data (products, users, orders) is securely stored in MongoDB. This project demonstrates building a complete, scalable, and production-ready eCommerce application from scratch, including both frontend and backend development, as well as payment integration and deployment.
Note-Taker Web-Application
Technologies
This project is a Note-Taker Web Application built using the MERN stack — MongoDB, Express.js, React.js, and Node.js. It allows users to securely create, edit, delete, and organize personal notes with an intuitive and responsive interface. User authentication is implemented using JWT and bcrypt to ensure that each user’s notes remain private and accessible only to them. The backend is powered by Node.js and Express.js REST APIs, with MongoDB as the database for efficient storage and retrieval of notes. Designed with a clean UI and smooth user experience, this application demonstrates full-stack development technologiess, secure authentication, and CRUD functionality.
